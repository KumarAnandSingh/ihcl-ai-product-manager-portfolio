## 🛠️ **Development Setup**

### **Prerequisites**
- Node.js 18.0+ with npm/yarn
- Python 3.10+ with pip/conda  
- Git with LFS support

### **Quick Start**
```bash
# Clone the repository
git clone https://github.com/your-username/ihcl-ai-product-manager-portfolio
cd ihcl-ai-product-manager-portfolio

# 🌟 VirtualAgent Platform
cd virtualagent-platform-dashboard
npm install && npm run dev  # → http://localhost:3002

# 📱 TelecomAssist Platform  
cd ../telecom-ai-nextjs
npm install && npm run dev -- --port 3005  # → http://localhost:3005

# 🛡️ Security Operations Platform
cd ../security-operations-nextjs  
npm install && npm run dev -- --port 3006  # → http://localhost:3006

# 🔧 Python Backend Services
cd ../my-telecom-ai-agent
python -m venv demo_venv && source demo_venv/bin/activate
pip install -r requirements.txt && python web_app.py  # → http://localhost:8000
```

### **Environment Configuration**
```bash
# Required API Keys (add to .env.local)
OPENAI_API_KEY=your_openai_key_here
ANTHROPIC_API_KEY=your_claude_key_here
DEEPGRAM_API_KEY=your_deepgram_key_here  # Optional: for advanced voice
ELEVENLABS_API_KEY=your_elevenlabs_key_here  # Optional: for premium voice
```

## 🚀 **Quick Start Guide**

### 🛠️ **System Requirements**
- **Node.js 18+** with npm/yarn package management  
- **Python 3.10+** with virtual environment and pip  
- **Git LFS** for large file support  
- **Modern Browser** with ES2022 support for frontend platforms  

### ⚡ **One-Command Launch**
```bash
# Clone and initialize the complete portfolio
git clone https://github.com/your-username/ihcl-ai-product-manager-portfolio
cd ihcl-ai-product-manager-portfolio

# Launch all platforms simultaneously
./launch-portfolio.sh

# OR manual platform startup:

# 1. VirtualAgent Platform (Port 3002)
cd virtualagent-platform-dashboard && npm install && npm run dev

# 2. TelecomAssist Platform (Port 3005)  
cd telecom-ai-nextjs && npm install && npm run dev -- --port 3005

# 3. Security Operations (Port 3006)
cd security-operations-nextjs && npm install && npm run dev -- --port 3006

# 4. Python Backend Services (Port 8000)
cd my-telecom-ai-agent && python -m venv demo_venv
source demo_venv/bin/activate && pip install -r requirements.txt
python web_app.py
```

### 🔧 **Environment Configuration**
```bash
# Create .env.local in each Next.js project directory
```
